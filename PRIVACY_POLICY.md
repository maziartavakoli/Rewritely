# Privacy Policy for Rewritely

Last updated: April 9, 2026

Rewritely is a browser extension that helps users rewrite selected or pasted text using a user-configured AI API.

## What data Rewritely handles

Rewritely may process:

- Text you explicitly select on a web page for rewriting
- Text you paste into the extension popup
- Custom instructions and saved prompts you create
- API configuration you provide, including base URL, model name, and API key

## How data is used

Rewritely uses this data only to provide its core functionality:

- Sending selected or pasted text, together with the chosen prompt or instruction, to the AI API endpoint you configure
- Streaming the rewritten result back into the page or popup
- Saving your settings and prompts locally so the extension works as expected

## Where data is stored

Rewritely stores the following locally in browser extension storage:

- API base URL
- API key
- Model name
- Saved prompts
- Default prompt selection

## Data sharing

Rewritely shares user-submitted text and the selected prompt or instruction only with the AI API endpoint configured by the user. Rewritely does not sell user data. Rewritely does not share user data with third parties for advertising, profiling, or any purpose unrelated to the extension's core functionality.

## Data retention

Locally stored settings and prompts remain in extension storage until you change them or remove the extension. Text selected or pasted for rewriting is sent only as needed to complete the requested rewrite.

## User control

You control what text is submitted. You can:

- Change or delete saved prompts
- Update or remove your API settings
- Stop using the extension at any time by disabling or uninstalling it

## Security

Rewritely stores settings in browser extension local storage. API requests are made from the extension, not from page scripts. However, you are responsible for choosing and trusting the AI API provider and endpoint you configure.

## Third-party services

Rewritely relies on the external AI API endpoint configured by the user. That provider's privacy practices and terms apply to any data sent to that service.

## Sensitive data notice

Users should not submit passwords, secrets, or other highly sensitive information unless they intend to send that content to their configured AI provider for processing.
